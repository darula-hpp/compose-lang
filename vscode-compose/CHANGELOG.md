# Change Log

## [0.1.0] - 2024-01-24

### Added
- Initial release
- Syntax highlighting for Compose language
- Code snippets (model, feature, guide, import)
- Build command integration
- Language configuration (comments, brackets)
- Custom file icon for .compose files

### Keywords Supported
- `model` - Data structures
- `feature` - Application behavior
- `guide` - Implementation details
- `import` - File imports

### Types Supported
- Primitives: text, number, bool, date, timestamp
- Rich types: image, file, markdown, json
- Modifiers: list of, optional (?)
- Enums: "value1" | "value2"

### Special Operators
- `@` - Reference external code (@reference/file.py)
